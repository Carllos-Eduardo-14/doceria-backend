# doceria-backend
Sistema de doceria com pedidos, pagamento e WhatsApp

# 🍰 Sistema de Doceria

Projeto backend desenvolvido em PHP com MySQL.

## 📌 Funcionalidades
- Cadastro de clientes
- Login
- Produtos
- Carrinho
- Pedido com entrega
- Cálculo de frete por km
- Pagamento (dinheiro, cartão, pix)
- Integração com WhatsApp
- Avaliações
- Favoritos

## 🗄️ Banco de dados
📥 [Clique aqui para baixar o banco de dados](./doceria.sql)

## 🚀 Como rodar o projeto
1. Importar o arquivo doceria.sql no phpMyAdmin
2. Rodar em servidor local (XAMPP/WAMP)
3. Acessar via navegador

## 👨‍💻 Tecnologias
- PHP
- MySQL
- HTML

## 📱 Extra
Sistema envia pedido automaticamente para o WhatsApp da loja.
